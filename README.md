# Structuring Planet Data 🪐

This project was completed as part of the µLearn Web Development learning path.

## Overview

A structured HTML table presenting information about planets in our solar system. The project focuses on organizing complex data using semantic HTML table elements and accessibility best practices.

## Features

- Semantic HTML table structure
- Table caption for better accessibility
- Use of table headers (`<th>`)
- Row grouping with `rowspan`
- Column grouping with `colspan`
- Organized planetary data presentation
- Clean and readable layout

## Technologies Used

- HTML5
- CSS3
- GitHub Pages
